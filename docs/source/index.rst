
jetbot_widgets
=====================================

Version: |release|

Jupyter widgets for NVIDIA JetBot


Quickstart
----------

To get started with jetbot_widgets, install with pip::

    pip install jetbot_widgets

or with conda::

    conda install jetbot_widgets
